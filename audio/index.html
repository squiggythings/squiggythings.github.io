Tools & Systems

WaveTracker

Wave tracker is a standalone audio tool and synthesis engine that I created to explore new ways of making music. It features a unique interface where you can draw waveforms directly onto a canvas to create sounds, and then manipulate the sound with various effects and modulation options. It's a fun and intuitive way to create digital sounds without needing much prior knowledge of music theory or synthesis.

It is written entirely in C#, with a custom built audio and DSP engine of up to 24 independent channels. Each channel has the capability to act as a sampler, wavetable synth, or noise generator.

Since its release, WaveTracker has been used by a variety of musicians and game developers to create unique sounds and music for their projects. It has gained a reputation for being one of the more intuitive tools in the chiptune music software space and it's also open source, so anyone can contribute to its development or customize it for their own needs.

WaveTracker - Visualizer

WaveTracker features a visualizer that provides real-time feedback on the audio output. The visualizer displays the waveform of each channel, as well as a piano roll that shows the notes being played. This allows users to see how their drawn waveforms and modulation settings are affecting the sound, making it easier to understand and refine their creations.

Wavetracker is often touted as having one of the best visualizers in the chiptune music software space, due to it's oscilloscopes and a piano roll being built in to the software itself, rather than forcing users to rely on third party software for visual feedback.

Gamepad Haptics Editor

The haptics editor I built for Insignia allows designers to easily author and preview controller haptics patterns. It features a custom envelope editor to arrange complex motor behavior, and a real-time preview mode that sends the haptics data to the controller as you edit.
It was built into Unity using C#, and the haptics data is sent to the controller using Rewired.

Since modern controller haptics are taken care of via direct audio signals, this tool allows backwards compatibility with older controllers by pairing the haptics event with dual motor envelopes.

2D Ambience Controller

For Insignia, I created a custom 2D ambience controller to allow designers to easily author how background ambience tracks play. Since ambiences are attached to the scene, this tool allows for the scene to "float" between two ambience tracks depending on the listener's position between the two nodes. This is incredibly useful for scenes that transition between two distinct in-game biomes.

The system has the added benefit of continuing the current ambience track if the next scene is part of the same environment, resulting in a completely seamless listening experience.

In this example the designer has created a smooth transition between a lush forest ambience and a windy valley ambience depending on the listener's altitude.

Enemy-Vocalization Manager

Insignia's enemies have various vocalizations that play in different situations, such as when they spot the player, attack, or die. I built a custom vocalization manager that automatically silences already playing vocalizations when a new one needs to play. This preserves the clarity of important vocalizations, such as attack telegraph vocals that plays when an enemy spots the player, which is crucial for gameplay feedback.

This entire system is completely hands off, requiring no manual intervention from designers once set up.Any sounds attached to the enemy's game object are automatically managed by the system.

Fish Synth

Fish Synth is an educational, real-time interactive synthesis and DSP system built in C#; with the purpose of exploring the fundamentals of sound design and music theory through interaction. It features a custom FM synthesis engine with ADSR envelope control for real-time sound generation, alongside a modular DSP effects chain including distortion, delay, and bitcrushing with reorderable signal routing.
The system is driven by an interactive control interface that allows real-time modulation of synthesis and effect parameters, as well as a lightweight event-based sequencing model for arranging and triggering notes with immediate audio feedback.

Abyss Demo - Dynamic Soundscape

A short platformer made to showcase adaptive music & audio systems. I drew all the graphics and programmed the gameplay, but this project is mainly meant to show off a dynamic soundscape and music system.
Here, I created a set of C# components to allow designers to easily author spatial ambience sounds and slightly randomize sound events. There is also an adaptive music system that changes the arrangement based on game progress. Notice how the music grows fuller the higher the player climbs.
