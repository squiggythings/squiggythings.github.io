Preferences


The preferences dialog can be accessed through the toolbar. It has settings that control the application behavior.

General

Setting Description
Frame Editor
Show row numbers in hex Display row numbers in hexadecimal instead of decimal
Show note off and release as text Display note off and release events as OFF and REL
Fade volume column Fade items in the volumn column according to their value. Lower volumes will appear more faded out.
Ignore step when moving Ignores the step value in edit settings when moving the cursor, will only use it when inputting notes.
Key repeat Enables key repetition when inputting notes and values
Restore channel state on playback If this is enabled, the application will reconstruct the state of each channel from previous frames upon playing.
Page jump amount The number of rows to jump when moving the cursor with the scroll wheel or PgUp/PgDown
Metering
Oscilloscope mode Determines the look of the oscilloscope in the song panel.
Mono: Displays a single oscilloscope
Stereo-split: Displays two oscilloscopes for the left and right channels respectively
Stereo-overlap: Displays a single oscilloscope with the left and right channels overlapped on each other, highlighting the difference between them
Meter decay rate Determines how responsive the volume meters in the song panel are. Fast is bouncier but not very consistent; slow is more consistent but less responsive.
Meter color mode Determines the look of the volume meters in the song panel. Flat is one shade of green; gradient gradually turns toward red the louder the signal is.
Flash meter red when clipping If this option is enabled, the volume meters will turn red whenever the signal is above 0db.

Samples/Waves

Setting Description
Sample settings
Normalize samples on import Automatically set a sample's volume to be as loud as possible when importing
Trim sample silence on import Automatically trim any silence before and after a sample when importing
Resample to 44.1 kHz on import If a sample is not 44.1 kHz, resample it to preserve its pitch
Preview samples in browser Plays audio files upon clicking on them in the sample browser
Include samples in visualizer by default If this is enabled, samples will be included in the piano roll visualizer by default
Default base key The base MIDI note value of any new sample. 48 (C-4) is the default
Resampling
Default wave resample mode Determines the default resampling algorithm that waves in new songs will have
Default sample resample mode Determines the default resampling algorithm that new samples will have when imported

Visualizer

Setting Description
Visualizer: Piano
Note speed Determines how fast notes scroll by in the piano roll. Lower values are slower
Change note width by volume If this is enabled, notes get thinner as they get softer
Change note opacity by volume If this is enabled, notes fade out as they get softer
Visualizer: Oscilloscopes
Wave zoom Determines how far zoomed in the waves in the oscilloscope are
Colorful waves If this is enabled, each oscilloscope window will use the same color as their notes in the piano roll; Otherwise, they are white.
Wave thickness Determines the thickness at which the waves are drawn in the oscilloscope
